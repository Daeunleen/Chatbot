한밭대학교 무선인터넷 안내
자세한 내용: https://www.hanbat.ac.kr/kor/sub06_080103.do
문의: S1동(도서관) 605호 정보통신망팀
전화: 042-821-1873, 1451

[무선인터넷 종류]
1. Hanbat_WLAN : 본교 학생 및 교직원용 (인증 필요, 학내 시스템 사용 가능)
2. Hanbat_WLAN_Guest : 인증 없이 인터넷만 가능 (학내 시스템 접속 불가, 속도 제한 있음)
3. Hanbat_WLAN_Eduroam : Eduroam에 가입된 타 대학 구성원용 (한밭대 구성원도 타 대학에서 사용 가능)

[사용자 ID 규칙]
- 학부생: 01 + 학번 (예: 학번 20181001 → ID: 0120181001)
- 대학원생: 02 + 학번 (예: 학번 20181001 → ID: 0220181001)
- 교직원: 교직원번호
- 비밀번호는 통합학사정보시스템과 동일
- Eduroam 접속 시 ID는 [ID]@hanbat.ac.kr 형식 (예: 201800001@hanbat.ac.kr)

[유의사항]
- 1계정당 최대 2대 기기 연결 가능
- WPA2 보안 방식 사용 → 구형 장비(특히 Windows XP)는 최신 드라이버 또는 패치 필요
- Android 10 이상은 ‘2단계 인증 없음’ 선택 불가 → ‘GTC’로 설정 필요
- 정보통신기반보호법 위반 시 별도 통보 없이 접속 제한 가능

[Windows 노트북 접속 방법]
1. 학교 홈페이지에서 Agent 설치
2. 설치 후 Hanbat_WLAN 또는 Hanbat_WLAN_Eduroam 선택
3. 클라이언트 실행 후 ID/PW 입력
   - Eduroam은 [ID]@hanbat.ac.kr 형식 사용

[iPhone / iPad 접속 방법]
1. 설정 → Wi-Fi → Hanbat_WLAN 선택
2. 다음 정보 입력
   - 이름(SSID): Hanbat_WLAN
   - 보안: WPA2 (기업용)
   - 사용자 이름: 위의 ID 규칙 참조
   - 암호: 통합학사정보시스템 비밀번호
3. 상단의 '연결' 클릭 후 '승인'

[Android 9 이하 버전 접속 방법]
1. Wi-Fi 설정 → Hanbat_WLAN 선택
2. 다음 정보 입력
   - 보안: 802.1x EAP
   - EAP 방식: PEAP
   - 2단계 인증: 설정 안 함
   - 사용자 이름: ID 규칙 참조
   - 비밀번호: 통합학사정보시스템 비밀번호
3. 연결 클릭

[Android 10 이상 버전 접속 방법]
1. Wi-Fi 설정 → Hanbat_WLAN 선택
2. 다음 정보 입력
   - EAP 방식: PEAP
   - 사용자 이름: ID 규칙 참조
   - 비밀번호: 통합학사정보시스템 비밀번호
   - CA 인증서: 없음 또는 사용 안 함
   - 고급 설정 → 2단계 인증 → GTC 선택
3. 저장 후 연결
